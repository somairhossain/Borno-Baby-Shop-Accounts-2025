# üßæ System Overview

This accounting system is built on the **double-entry bookkeeping framework**, ensuring that:

* Every transaction affects at least two accounts
* Total Debits always equal Total Credits
* Financial statements are generated from structured ledger balances

The system includes:

* Chart of Accounts (COA)
* Journal Entries
* General Ledger
* Trial Balance
* Adjusted / Final Trial Balance
* Notes to Accounts
* Income Statement
* Balance Sheet
* Cash Flow Statement
* Equity Statement
* Depreciation Schedule
* Salary Sheet
